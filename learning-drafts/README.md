# Learning Drafts

This folder stores pasteable Rust learning drafts.

Each draft should read like a short paper for humans, while still being safe to paste into Rust source code.

Format rule:

```text
explanations are written as // comments
real Rust code stays below or beside the explanation
hardware/CPU meaning comes before Rust syntax meaning
```

Why this format works:

```text
the file teaches the idea
the compiler ignores the teaching comments
the same file can become src/main.rs when the bite is ready
```
